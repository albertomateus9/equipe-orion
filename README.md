# Orion

Projeto da equipe Orion para a Hacka 2026 da turma P1 de Informatica Subsequente da EETEPA Vilhena Alves.

## Ideia

Painel com informacoes em tempo real sobre entrada e saida de alunos e alocacao das turmas.
